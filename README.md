# Smart Campus Event Management System

## 📌 Overview

A full-stack web application designed to manage campus events efficiently.
The system allows users to create, manage, and participate in events while providing administrators with tools for monitoring, reporting, and resource allocation.

---

## 🚀 Features

### 👤 User Features

* User registration and login
* Role-based access (Student, Faculty, Admin)
* Browse and search events
* Register for events
* Submit feedback

### 🛠️ Admin & Faculty Features

* Create, edit, and cancel events
* Assign resources (rooms, equipment)
* Send notifications
* View reports and analytics

### 🔒 Security Features

* Role-Based Access Control (RBAC)
* Password hashing
* Audit logging
* Data protection design
